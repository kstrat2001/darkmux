Connecting to the daemon

The viewer fetches flow records from a local darkmux serve daemon. To start it:

darkmux serve

Default bind is 127.0.0.1:8765. Use --port or --bind to change. The viewer auto-detects a running daemon on load; if you don't see "daemon · live", refresh after starting.

For a file-based session (no daemon needed), use the Load .jsonl button instead.
